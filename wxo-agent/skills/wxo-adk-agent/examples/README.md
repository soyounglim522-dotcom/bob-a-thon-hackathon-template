# Examples

이 폴더에는 wxo ADK 예제들이 포함되어 있습니다.

원본 예제는 GitHub에서 확인할 수 있습니다:
https://github.com/soyounglim522-dotcom/bob_wxo_hackathon/tree/main/skills/wxo-adk-agent/examples

| 예제 디렉토리 | 패턴 | 설명 |
|---|---|---|
| `personal_banking/` | Multi-tool collaborator | 계좌 조회, 이체, 연락처 관리 |
| `customer_care/` | Manager + 두 개의 collaborator | 고객 케어 + ServiceNow 분리 구조 |
| `customer_care_planner/` | Planner 스타일 agent | Planner 모드 + format_task_results 패턴 |
| `healthcare_provider/` | OpenAPI tool (Python 불필요) | OpenAPI spec 직접 import |
| `ibm_knowledge/` | Knowledge-base agent | 벡터 KB + stock_price 툴 조합 |
| `agentic_memory/` | Agentic memory / 티켓 생성 | 최소 구조, 학습용 |
| `voice_enabled_elevenlabs/` | Voice channel | ElevenLabs TTS 연동 |

각 예제를 복사해서 `../my-wxo-agent/` 아래 팀 작업 폴더로 사용하세요.
